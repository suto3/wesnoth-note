 261     # The first_time_only moveto event causes a trident *image* to
 262     # show up on the hex, and causes tridents to be explained. The
 263     # moved unit gets the option to take it.
 264     #
 265     # If the unit takes it, a trident *object* is put on the hex, and
 266     # the unit gets it because he is standing there already.
 267     #
 268     # If the unit does not take it, another moveto event is
 269     # added. This one is *not* first_time_only. It does pretty much
 270     # the same thing, but with shorter messages.
 271     #
 272     # If the unit does not take the trident, an [allow_undo] is added
 273     # to the event.
