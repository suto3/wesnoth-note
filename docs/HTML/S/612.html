 102         /**
 103          * Sets a child in the grid.
 104          *
 105          * When the child is added to the grid the grid 'owns' the widget.
 106          * The widget is put in a cell, and every cell can only contain 1 widget if
 107          * the wanted cell already contains a widget, that widget is freed and
 108          * removed.
 109          *

 111          * @param row                 The row of the cell.
