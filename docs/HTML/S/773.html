  24 /**
  25  * Determines if a string might contain variables to interpolate.
  26  * This can allow one to skip future interpolations (plural -- if there is only
  27  * one interpolation, the savings are not worth this check). In this spirit,
  28  * precision is sacrificed in the name of efficiency; the check is quick and
  29  * allows false positives, but there are no false negatives. (A false negative
  30  * would lead to incorrect behavior, whereas a false positive leads to merely
  31  * inefficient behavior.) In practice, false positives should be uncommon enough
  32  * to not worry about.
  33  */
  34 inline bool might_contain_variables(const std::string &str)
  35 { return str.find('$') != std::string::npos; }
