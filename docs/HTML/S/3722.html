 185     text= _ "The editor features two separate modes of operation:" + _ "
 186 
 187 <header>text='Pure Map Mode'</header>" + _ "
 188 
 189 Allows only the composing of the terrain map itself and the definition of leader starting positions." + _ "
 190 How the information is saved depends on the loaded file:
 191 
 192 <bold>text='Native'</bold>" + _ "
 193 A new map or file containing only the arguments to the map_data attribute.
 194 
 195 The produced map can be played in the “User Maps” game type at the create multiplayer game dialog if saved to the default directory.
 196 
 197 <bold>text='Embedded'</bold>" + _ "
 198 Scenario files containing a valid map_data attribute (not a file include) will be opened in this submode. The editor replaces only the content of map_data and leaves everything else in the scenario untouched. Maps opened this way are marked [e] in the Maps menu." + _ "
 199 
 200 <header>text='Scenario Mode'</header>" + _ "
 201 
 202 The Scenario mode allows several extra tools to be used, such as the Unit tool. At least one side must be defined in order to use these tools, however.
 203 
 204 In this mode, terrain data is stored in the map_data attribute and saved into a file with any applicable WML."
