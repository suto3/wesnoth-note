  96          *
  97          * Setting the selection range will re-position the cursor depending on the
  98          * selection direction, specified by the length's sign. Selecting beyond the
  99          * start or end of the text is safe; the final selection will be limited
 100          * accordingly.
 101          *
 102          * @note The range extents are measured in Unicode characters, not bytes.
 103          *       Using byte offsets may produce unexpected results depending on the
 104          *       text contents.
 105          *
 106          * @param start               Start offset, in characters.
 107          * @param length              Selection length, in characters. If zero, the
 108          *                            current selection is canceled. If negative, the

 111          *                            direction as well; otherwise the selection and

 303         /**
 304          * Every key can have several behaviors.
 305          *
 306          * Unmodified                 No modifier is pressed.
 307          * Control                    The control key is pressed.
 308          * Shift                      The shift key is pressed.
 309          * Alt                        The alt key is pressed.
 310          *
 311          * If modifiers together do something else as the sum of the modifiers
 312          * it's listed separately eg.
 313          *
 314          * Control                    Moves 10 steps at the time.
 315          * Shift                      Selects the text.
 316          * Control + Shift            Inserts 42 in the text.
 317          *
 318          * There are some predefined actions for results.
 319          * Unhandled                  The key/modifier is ignored and also reported
 320          *                            unhandled.
 321          * Ignored                    The key/modifier is ignored and it's
 322          *                            _expected_ the inherited classes do the same.
 323          * Implementation defined     The key/modifier is ignored and it's expected
 324          *                            the inherited classes will define some meaning
 325          *                            to it.
 326          */
